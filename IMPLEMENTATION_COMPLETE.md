## 🐛 故障排除

### Q: 圖片沒有顯示藝術風格？

A: 檢查以下步驟：
```bash
# 1. 驗證環境
python3 scripts/check_env.py

# 2. 檢查瀏覽器控制台 (F12)
#    - 是否有紅色錯誤？
#    - API 返回什麼狀態碼？

# 3. 清空快取並刷新
rm -rf .cache/processed-images/
```

### Q: 為什麼圖片加載這麼慢？

A:
- **首次訪問**: 1-3 秒是正常的 (包含下載和處理)
- **後續訪問**: 應該 <50ms (快取)

如果始終很慢，檢查：
```bash
# CPU 使用率
top

# 網絡
curl -w "時間: %{time_total}s\n" https://example.com/image.jpg
```

### Q: 如何禁用圖片處理？

A: 注釋掉 `src/app/login/page.tsx` 中的這部分：

```typescript

setImagePool(sourcePool);  // 而不是 processedImages
```

## 🎓 技術亮點

### ✨ 藝術風格算法

```python
# 多步驟轉換過程
1. 邊緣保留平滑 (dtFilter)
   └─ 消除微小雜訊，保留主要特徵

2. K-means 色塊化 (k=24)
   └─ 將圖片簡化為 24 種顏色

3. 特徵線條提取 (adaptiveThreshold)
   └─ 找出輪廓線條 (頭髮、五官)

4. 合成 (bitwise_and)
   └─ 結合色塊和線條 → 最終藝術風格
```

### 🔋 性能優化

- **並行下載**: Promise.all() 同時處理 15 張圖片
- **智能快取**: MD5 哈希 + 30 天有效期
- **子進程模型**: 每個圖片獨立 Python 進程

### 🛡️ 錯誤處理

- 網路故障 → 回退到原始圖片
- 處理失敗 → 自動重試和回退
- 快取遺失 → 自動重新生成
